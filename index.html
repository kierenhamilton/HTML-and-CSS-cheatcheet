Getting started

First, you need to type the following things into a HTML document:

  1. <!DOCTYPE html>
  2. <html>
  3. ---<head>
  4. ---</head>
  5. ---<body>
  6. ---</body>
  7. </html>

Adding CSS

Firstly, you must add a CSS file in the same directory as the HTML file.

then, inside of the <head> tag you must add the following:

  • <link rel="stylesheet" href="NAME-OF-CSS-FILE.css">

This will apply the styling in your CSS document to your HTML website.

HTML basics

Tag Description
<h1> This is a header. Available headers are h1 through h8, each getting progressivly smaller.
<p> This lets you write paragraphs in your website
<ol> This creates an ordered list
<li> This lets you add individual elements inside of the list
<ul> This lets you create an unordered list (just like the ordered list but with bullet points instead of numbers)
<div> This creates a section to help you organize your website
